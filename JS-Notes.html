INTRODUCTION

Javascript is a computer programming language that enables you to design interactive web pages.
It was created by Brenden Eich in 1995 and may also be referred to as:

  • LiveScript
  • Mocha
  • ECMAScript
  • JScript
Syntax:
                    
    var x = 10;
    var y = 30;
    
    function add( arg1, arg2 ) {
        return arg1 + arg2;
    }
    add( x, y);
                    
                
Log to Console:
                    
    var num = 10;
    console.log( "string", 10.11, true, false, null, num );

General Terminology

Encapsulation is grouping data/information together
Nouns and Adjectives translate to properties.
Verbs translate to functions.
Objects can embedded within other objects
Properties are enclosed with braces { }.
A method is a function or subroutine within an object.

API ( Application Programming Interface ) is a collection of functions and properties.

STRINGS & VALUES

A string can be defined using single or double quotes; a semicolon is used to define the end of a string.
To nullify a character, place a backslash before the character you wish to nullify or use the opposite type of quotation mark:

                    
    “She said “hello”” x
    “She said \”hello\”” ✓
    “She said ‘hello’ “ ✓
    ‘She said “hello” ‘ ✓

Numbers can be: whole integers ( ex. 1, 50, 200 ) or floating points ( ex. 0.56, 100.71 )
Boolean is a data type that presents only two options ( true or false )
Null and undefined translate to “empty” or “void” ~ there is no value
NaN ( Not a Number ) allows you to learn if a certain type of variable can be inputted into a specific function.

VARIABLES

To create a variable:

  • Type var
  • Type the name you wish to refer to the variable as ( w/out quotes )
  • Press “=” on the keypad
  • Assign a string, numeric value, boolean, etc. to the variable
  • End with a semicolon
                    
        Ex. var  name = “Henry”;
            undefined 
            name 
            “Henry”

To replace a variable:

  • Type the name of the variable
  • Press “=” on the keypad
  • Assign a new value to the variable
  • End with a semicolon
                    
        Ex. var  name = “Henry”;
            undefined 
            name
            “Henry”
            name = “Sarah”;
            “Sarah”
            name
            “Sarah”

To store an unfinished variable:

  • Type var
  • Name the variable
  • Type a semicolon
  • Press “Enter” on the keypad
  • When ready to assign, type the name of the variable ( w/out quotes )
  • Press “=” on the keypad
  • Assign a value to the variable
                    
        Ex. var  name;
            undefined
            name = “Henry”;
            “Henry”
            name
            “Henry”

Constants:

  • Created in the same way as a variable, but type const instead of var
  • Once a value is assigned, it can never be replaced as long as the program is still in progress
  • Cannot be stored like variables

A symbols table is a database that stores all of the symbols/variables that allow you to access values
( located in the window object ) ~ like a giant warehouse

OPERATORS

An operator is a predefined function ( a function defined in the JIT compiler )
Visit this Operator Precedence Website for more information

Order of precedence of the arithmetic operators, remember B O DM AS:

  • Brackets: ( )
  • pOwers: function call for exponent
  • Division or Multiplication: / or *
  • Addition or Subtraction: + or -
  • Remainder/Modulus: % ( gives you remainder of division problem )

Editing a variable value using arithmetic operators:

  • += will add the proceeding number to the existing variable value
  • -= will subtract the proceeding number from the existing variable value
  • *= will multiply the proceeding number by the existing variable value
  • /= will divide the proceeding number from the existing variable value
  • %= will give you the remainder of dividing the proceeding number from the existing variable value
  • **= will raise the existing variable value to the power of the proceeding number

The + can also be used when combining strings:

                    
    Ex. “Henry” + “ is nice”
    “Henry is nice"

Polymorphism is an organism or inorganic object or material which takes various forms.

                    
    Ex. “Henry” + 10 + true
    “Henry10true”
CODING

In an HTML file, anything written between the script tags ( <script> </script> ) will be classified as Javascript.

Display a string in the console:
In between the script tags, type console.log - Create parenthesis next to the function -
Within the parenthesis, type the text you wish to display as a string

                    
    Ex. <script> console.log( “Hello world” ) </script>

You can log out strings, numeric values, variables, boolean, null/undefined, etc.
You can log out multiple values in one line by separating them with commas
But normally, we will just use a separate .js file for our javascript code.
To connect a .js file with a .html file, use this:

                    
    <script src = ”_file_”></script>

**Make sure to put this line of code at the end of your code

FUNCTIONS

Function or subroutine is task defined by set of instructions that can be run at certain point in time.
Execution context is a list of instructions to complete the routine.

To create a function:

  • Type function
  • Name the function
  • Create a set of parentheses
  • Within the parenthesis, define function’s parameters ( variables that can be assigned values )
  • On the next line, add a pair of curly brackets
  • Within the curly brackets, begin listing the instructions
  • If necessary, you can return the instruction by typing return _variable_


To actually invoke the function:

  • Type the name of the function in a separate line
  • Create a set of parenthesis
  • Within the parenthesis, type the arguments ( values that you assign to the parameters )
  • You can also change the arguments of a function in the console itself
                    
    function makeCoffee( sugar, milk ) {
            var instructions = "Boil water, ";
            instructions += "pour into cup, ";
            instructions += "add coffee granules, ";
            instructions += "add " + sugar + " spoons of sugar, ";
            instructions += "add " + milk + " % milk.";
            return instructions
        }
        
     console.log( makeCoffee( 2, 20 ) )
OBJECTS & ARRAYS

Objects are obvious :)
Arrays are a list of objects

To create an object, add a pair of curly brackets. Within the object, you can:
Create a property by adding key name and assigning a value ~ separate properties w/ commas
Create a function or subroutine that you want performed on or by the object

                    
    var car = {
        color: "red",
        speed: 200,
        drive: function(){ return "drive"; }
    };

To create an array, add a pair of square brackets.
Within the array, you can create elements; they will automatically be assigned to a number starting from 0 ~ use commas

                    
   var shoppinglist = [ "apples", "oranges", "bananas", "grapes" ];
    OR
   var shoppinglist = [
       "apples",
       "oranges",
       "bananas",
       "grapes",
    ];

When embedding a method in an object, assign the function a specific name.
When embedding a method in an array, don’t assign the function a specific name; it will automatically be assigned to a number.

                    
    var shoppinglist = {
        store: "giant",
        fruit: [ 
            "apples", 
            "oranges", 
            "bananas", 
            "grapes" ],
        dairy: [ 
            "cheese", 
            {food: "yogurt", 
            brand: "chobani", 
            flavor: ["keyLimeCrumble", 
                    "coconutSnowball", 
                    "almondCocoLoco"] } ]
        pay: function(){ return "pay"; }
    };
